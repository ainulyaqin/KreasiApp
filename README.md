# KreasiApp
Spring Boot Project


## How to run

1.  `$ git clone https://github.com/ainulyaqin/KreasiApp.git.`<br /> 
2. `$ mvn verify`<br /> 
3. `$ mvn spring-boot:run`<br /> 
4. Goto browser `http://localhost:8080/swagger-ui/index.html`<br /> 
5. H2 url `http://localhost:8080/h2-console/login.do`
